# Clues for Fools
The definitive speedrunning suite for [Clues by Sam](https://cluesbysam.com/).
This **Userscript** maximizes your board and sharpens visual cues, making leads impossible to miss and turning every solve into a record-breaking run.

## Features
* **Theme-Aware Highlights:** High-contrast colors for referenced names and professions across all game themes.
* **Expanded Layout:** Maximizes board size and increases padding so information has more room to breathe.
* **Muted Non-Clues:** Subtly dims non-essential text to help your eyes snap directly to actionable hints.

## Installation
Install via [Greasy Fork](https://greasyfork.org/en/scripts/567777-clues-for-fools) for updates, or manually add clues-for-fools.user.js to [Tampermonkey](https://www.tampermonkey.net/faq.php#Q102), [Violentmonkey](https://violentmonkey.github.io/guide/creating-a-userscript/), or [Greasemonkey](https://wiki.greasespot.net/Greasemonkey_Manual:Installing_Scripts).
